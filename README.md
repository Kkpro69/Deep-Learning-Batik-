# Deep-Learning-Batik-
Klasifikasi Batik Nusantara

Analisis Deep Learning pada batik nusantara menggunakan Vision Transformer (ViT) bertujuan untuk melihat apakah dengan menggunakan ViT hasil dari klasifikasi batik yang tidak banyak dapat menghasilkan klasifikasi yang akurat.

Menggunakan ViT yang notabennya perlu data yang sangat banyak, pada kasus ini menghasilkan akurasi yang cukup baik di mana hasilnya 96%. Kesimpulannya adalah menggunakan ViT sangat cocok untuk klasifikasi gambar namun perlu data yang banyak agar model belajar lebih baik lagi.
